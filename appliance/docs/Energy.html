NIWA Systems : Energy

Energy modelling

Energy intensity is derived from census demographic data for census meshblocks. The schema is designed to be generic and to allow some form of aggregation to be applied to a set of model equation parts, each which derives information from a single census demographic statistic.

Physical model

Model meta data

The meta data needed to configure a energy model run.

activity

The energy activity to model energy intensity for, eg. household heating, total energy consumption

  • id (SERIAL)
  • code (VARCHAR(32))
  • description (TEXT)

scenario

The scenario to apply to modelling energy intensity, eg. current accounts, continuity

  • id (SERIAL)
  • code (VARCHAR(32))
  • description (TEXT)

Model definition

A model definition provides the aggregation rules to use for producing an energy intensity value for a specific energy producing activity and scenario.

   Energy Intensity = some demographic data * coefficient + POW (some other demographic data, coefficient)
                      |---- model definition part 1 ----| + |---------- model definition part 2----------|
                      +------------------------- complete model definition ------------------------------+

model_definition

Hold information about a specific model

  • id (_SERIAL)
  • activity_id (INTEGER)
  • scenario_id (INTEGER)
  • number_parts (INTEGER)
  • description (TEXT)

model_definition_part

The individual parts of the equation that are aggregated to produce final intensity value.

  • id (SERIAL)
  • model_definition_id (INTEGER)
  • census_class_id (INTEGER)
  • equation (TEXT)
  • coefficient (NUMERIC)
  • part (INTEGER)

Model run

This is the output of a model when run on meshblock census demographic data. A stored procedure will take care of generating and executing the SQL statements that will derive the Energy intensity value for a specific energy activity and scenario.

intensity

  • id (SERIAL)
  • census_admin_area_id (INTEGER)
  • model_definition_id (INTEGER)
  • year (SMALLINT)
  • value (NUMERIC)
